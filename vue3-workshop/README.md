# Vue 3 学习工坊

系统学习 Vue 3 核心特性与最佳实践的交互式学习站点。

## 📚 知识点覆盖

### 响应式系统
- ref 与 reactive
- computed 计算属性
- watch 与 watchEffect

### 组合式 API
- setup 语法糖
- 生命周期钩子
- 组合式函数 (Composables)

### 组件通信
- Props 父传子
- Emits 子传父
- Provide / Inject

### 模板语法
- 指令与事件处理

### 高级特性
- Teleport 传送门
- Suspense 异步组件
- Transition 过渡动画
- 自定义指令

### 路由与状态
- Vue Router 路由系统
- Pinia 状态管理

### TypeScript
- TypeScript 集成

## 🎯 功能特性

- **渐进式学习**：从基础到进阶的 18 个核心知识点
- **代码示例**：每个知识点包含 3 个实用代码示例
- **测验题**：每个知识点附带测验题，巩固学习成果
- **闪卡复习**：支持翻转卡片、键盘快捷键的闪卡模式
- **学习进度**：本地存储学习进度，跟踪掌握情况
- **知识点索引**：支持搜索和分类筛选
- **PWA 支持**：可离线访问，支持添加到主屏幕

## 🚀 部署

本项目为纯静态站点，可部署到任何静态托管平台：

### Vercel（推荐）
```bash
vercel deploy
```
