为什么要学习前端性能优化？

前端性能直接影响用户体验和转化率。Google研究表明，页面加载时间从1秒增加到3秒，跳出率增加32%；加载时间超过3秒，53%的移动用户会离开页面。

Core Web Vitals（LCP、INP、CLS）已经成为Google搜索排名的重要信号，性能优化不仅是技术问题，也是SEO问题和商业问题。Amazon统计显示，页面每慢100ms，销售额下降1%。
